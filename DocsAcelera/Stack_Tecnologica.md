# Stack Tecnológica

## Backend
- **Linguagem**: Python
- **Framework**: FastAPI
- **ORM**: SQLAlchemy

## Frontend
- **Templates**: Jinja2

## Banco de Dados
- **Local**: SQLite
- **Integração**: MSSQL (ERP Senior)

## Bibliotecas
- **Processamento de Dados**: Pandas
- **Geração de Planilhas**: OpenPyXL
- **Integração SOAP**: Zeep

## Padrões Utilizados
- Arquitetura MVC (Model-View-Controller)
- Rotas RESTful
- Validação de Dados com Pydantic
