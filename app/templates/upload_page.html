Importar Clientes

Envie um arquivo Excel (.xlsx, .xls), CSV (.csv) ou JSON (.json) contendo os dados dos clientes.

Formatos Suportados

Excel (.xlsx, .xls)

Arquivos do Microsoft Excel com planilhas de clientes.

CSV (.csv)

Arquivos de texto com valores separados por vírgula.

JSON (.json)

Arquivos JSON com array de objetos de clientes.

Campos Reconhecidos

O sistema reconhece automaticamente os seguintes campos (com suas variações):

Nome: name, customer_name, full_name, client_name
Email: email, customer_email, email_address
Telefone: phone, phone_number, telephone
Empresa: company, company_name, organization
Endereço: address, street_address
Cidade: city
Estado: state
País: country
CEP: postal_code, zip_code, zip, postcode
Tipo: customer_type, type, category
Receita: total_revenue, revenue, total_sales
