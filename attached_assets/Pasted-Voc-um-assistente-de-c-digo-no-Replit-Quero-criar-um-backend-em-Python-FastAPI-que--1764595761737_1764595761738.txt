   ```python
       pelo menos, os campos:
         - cpf

         - cargo

         - data_demissao

         - centro_custo
       A query pode ser deixada com TODO para eu ajustar depois.

   ```

   Exemplo de estrutura de query (NÃO assuma nomes reais, deixe claro em comentários que são placeholders):

   ```sql
   SELECT
       f.CPF as cpf,
       f.NOME as nome,
       f.FUNCAO as funcao,
       f.CARGO as cargo,
       f.DT_ADMISSAO as data_admissao,
       f.DT_DEMISSAO as data_demissao,
       emp.NOME as empresa_nome,
       un.CNPJ as cnpj_unidade,
       un.NOME as nome_unidade,
       un.CENTRO_CUSTO as centro_custo
   FROM FUNCIONARIOS f
   JOIN EMPRESAS emp ON emp.ID = f.EMPRESA_ID
   JOIN UNIDADES un ON un.ID = f.UNIDADE_ID
   WHERE emp.NOME = 'TELOS'
   ```
