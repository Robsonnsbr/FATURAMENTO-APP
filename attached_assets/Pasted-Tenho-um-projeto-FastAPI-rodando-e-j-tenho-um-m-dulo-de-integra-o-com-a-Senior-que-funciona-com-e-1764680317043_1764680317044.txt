O código desse módulo já está pronto e usa esta SQL base para faturamento (já adaptada para usar o banco correto):

SELECT
    R034FUN.NUMCAD,
    R034FUN.NOMFUN,
    R034FUN.DATADM,
    R034FUN.CODCCU,
    R034FUN.DATAFA,
    R034FUN.VALSAL,
    R034FUN.SITAFA,
    R010SIT.DESSIT,
    R024CAR.TITRED,
    R044CAL.PERREF,
    R046VER.CODCAL,
    R046VER.CODEVE,
    R008EVC.DESEVE,
    R046VER.REFEVE,
    R046VER.VALEVE
FROM
    [opus_hcm_221123].dbo.R034FUN
JOIN
    [opus_hcm_221123].dbo.R046VER ON
        R034FUN.NUMEMP = R046VER.NUMEMP AND
        R034FUN.TIPCOL = R046VER.TIPCOL AND
        R034FUN.NUMCAD = R046VER.NUMCAD
JOIN
    [opus_hcm_221123].dbo.R024CAR ON
        R034FUN.ESTCAR = R024CAR.ESTCAR AND
        R034FUN.CODCAR = R024CAR.CODCAR
JOIN
    [opus_hcm_221123].dbo.R010SIT ON
        R034FUN.SITAFA = R010SIT.CODSIT
JOIN
    [opus_hcm_221123].dbo.R008EVC ON
        R046VER.TABEVE = R008EVC.CODTAB AND
        R046VER.CODEVE = R008EVC.CODEVE
JOIN
    [opus_hcm_221123].dbo.R044CAL ON
        R046VER.CODCAL = R044CAL.CODCAL
WHERE
    '{periodo}' BETWEEN R044CAL.INICMP AND R044CAL.FIMCMP
    AND R034FUN.NUMEMP = {numemp}
    -- filtros opcionais de CODCCU, CODCAL, SITAFA
ORDER BY
    R034FUN.NOMFUN,
    R046VER.REFEVE;

1) Criar um router FastAPI para integração com a Senior

Crie um arquivo, por exemplo: app/routers/senior_billing.py com um APIRouter que exponha os seguintes endpoints:

GET /integrations/senior/status

Usa get_connection_info() e test_connection()

Retorna JSON com:

api_domain

api_key_configured

database

numemp_telos

health (resultado de test_connection())

GET /integrations/senior/cost-centers

Usa fetch_cost_centers()

Retorna lista de { codccu, nomccu }.

GET /integrations/senior/employees

Usa fetch_employees_telos()

Retorna lista de funcionários TELOS.

GET /integrations/senior/billing/summary

Query params:

periodo: str (YYYY-MM-DD)

numemp: int (default = 6)

codccu: str | None

codcal: int | None

sitafa: int | None

Retorna JSON com:

total_lancamentos

periodo

numemp

codccu (ou “Todos”)

sitafa

GET /integrations/senior/billing

Mesmos query params de /billing/summary

Usa fetch_billing_data(...)

Retorna lista de registros (cada item com os campos já montados no módulo: numcad, nomfun, datadm, codccu, datafa, valsal, sitafa, dessit, titred, perref, codcal, codeve, deseve, refeve, valeve).

Inclua esse router no main.py (ou onde estiver a app FastAPI) com um prefixo adequado, por exemplo:

from app.routers import senior_billing
